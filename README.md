# Tableau

Tableau Data Visualisation Projects
Project Overview

This repository contains Tableau projects completed during my Data Technician Skills Bootcamp. The projects demonstrate my ability to explore different datasets, select appropriate visualisations and build interactive dashboards that communicate clear insights.

Spotify Data Dashboard

Analysed Spotify data using measures such as popularity, danceability, acousticness and tempo.
Compared music genres and artists using bar charts and scatter-style visualisations.
Used filters and interactive colour legends to explore specific genres and artists.
Created a combined visualisation comparing track popularity and tempo across genres.
Presented music data in a dashboard designed to highlight relationships between audio features.

Life Expectancy Dashboard

Compared average life expectancy across countries and continents.
Used line charts to examine changes in life expectancy over time.
Created scatter plots to investigate the relationship between BMI and life expectancy.
Applied country, continent and year filters to support interactive exploration.
Visualised population differences by gender using percentages and labelled charts.

Great Britain Employment Dashboard

Analysed changes in employment between 2011 and 2014 across Great Britain.
Created a geographic map to display employment changes by city.
Compared employment growth and decline across industries using bar charts.
Used proportional symbols, colour and labels to communicate the size and direction of change.
Added interactive industry and measure filters to help users investigate the data.
